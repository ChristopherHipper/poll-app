🗳️ Poll App

An interactive polling application where users can create polls, vote, and view live results – built using **TypeScript**, **SCSS**, and **Angular**.

🎯 Features

📊 Create custom polls with multiple options
🗳️ Vote in real-time
📈 Live result visualization
🎨 Clean and responsive UI
⚡ Fast performance powered by Vite
🛠️ Fully typed logic with TypeScript

🛠️ Tech Stack

* Angular V20.3.3
* TypeScript
* SCSS

🚀 Getting Started
```bash
# Clone the repository
git clone https://github.com/ChristopherHipper/poll-app.git

# Navigate into the project
cd poll-app

# Install dependencies
npm install

# Start development server
npm start
```
🏗️ Build
```bash
npm run build
```
